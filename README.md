THE ULTIMATE GAME SCORE TRACKER

ScoreKeep

ScoreKeep is a Flutter app for creating, managing, and tracking scores for your games. Works for cards, board games, party games - whatever you're playing. Keep score easily and check back on past sessions whenever you want.

Features

Game History: View and manage past game sessions
Dark Mode Toggle: Switch between light and dark themes
Persistent Storage: Sessions saved locally using custom storage service
Undo Delete: Accidentally deleted a game? Undo it from the snackbar
Animated UI: Smooth animations and transitions
Firebase Analytics: Track user actions and screen views
Pull-to-Refresh: Refresh your list of game sessions
Haptic Feedback: Responsive tactile interactions for buttons and actions


Tech Stack

Flutter 3.x
Firebase Analytics
Custom local storage service
Clean, reusable widgets
